# Análisis de Migración Intercomunal en Chile con Machine Learning
## Descripcion
Proyecto de análisis de datos y machine learning para estudiar los patrones de migración intercomunal en Santiago de Chile, combinando técnicas de ciencia de datos con una interfaz web interactiva para visualizar los resultados de los distintos modelos utilizados.


## Estructura del Proyecto

```
analisis-migracion-intercomunal-chile-ml/
│
├── ml_pipeline/              # Pipeline de Machine Learning
│   ├── DATASCIENCE_E2Codigo.ipynb  # Notebook principal de análisis
│   └── data/                 # Datasets utilizados
│
├── frontend/                 # Aplicación web (React + TypeScript + Vite)
│   └── ...                    # Código del frontend
│
└── presentation/             # presentación utilizada para exponer el proyecto.
```

## Características

- **Análisis Exploratorio de Datos (EDA)**: Visualización y comprensión de patrones migratorios
- **Modelos de Machine Learning**: Implementación de algoritmos predictivos
- **Dashboard Interactivo**: Interfaz web para explorar resultados
- **Visualizaciones**: Gráficos y mapas interactivos de flujos migratorios
